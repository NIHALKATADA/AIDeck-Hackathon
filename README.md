# 🤖 AIDeck | AI File Agent

An intelligent, AI-powered agent that instantly drafts and formats professional 16:9 PowerPoint presentations and Excel datasets from a single topic prompt. Built for speed and precision.

## ✨ Features
* **Zero-to-Deck:** Enter a topic and generate a fully structured presentation draft in seconds.
* **Smart Context Levels:** Choose between Concise, Standard, or Detailed text generation.
* **Live UI Preview:** Edit content and preview themes (Modern Circle, Split Accent, Top Banner) directly in the browser.
* **Native Export:** Downloads clean, native `.pptx` and `.xlsx` files ready for the boardroom.

## 🛠️ Tech Stack
* **Frontend:** Streamlit
* **AI Engine:** Groq API (Llama-3.3-70b-versatile)
* **File Generation:** `python-pptx`, `pandas`, `openpyxl`
* **Data Validation:** Pydantic
